# Building
- **Visual Studio 2015 required (I guess)**
- Rename *[REPO]/Twice/Constants.Auth.txt* to *Constants.Auth.cs*
- Replace *YOUR-CONSUMER-KEY* and *YOUR-CONSUMER-SECRET* with your access tokens in the txt file
- Open solution
- Build
